# TweetKeywordVisualizer

**TweetKeywordVisualizer** is a Python project that analyzes the most common keywords in English tweets and visualizes them using bar charts and word clouds.

> Note: This version uses sample/demo tweets. To fetch real Twitter data, you need a Twitter API key.

## Features

- Analyze top keywords from tweets
- Generate bar chart of the top 10 keywords
- Generate word cloud of all keywords
- Easy to extend for real Twitter API usage

## Installation

```bash
git clone https://github.com/Uranium227/tweetkeywordvisualizer.git
cd tweetkeywordvisualizer
pip install -r requirements.txt
python tweetkeywordvisualizer.py
